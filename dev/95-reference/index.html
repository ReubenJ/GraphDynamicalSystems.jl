GraphDynamicalSystems.QualitativeNetwork — Type
struct QualitativeNetwork

A qualitative network model as described in "Qualitative networks: a symbolic approach to analyze biological signaling networks".

  • graph: Graph containing the topology and target functions of the network

  • state: State of the network

  • N: The maximum activation level/state value of any component

Systems that include the model semantics wrap around this struct with an ArbitrarySteppable from DynamicalSystems. See aqn for an example.

GraphDynamicalSystems.build_qn_grammar — Method
build_qn_grammar(
    entity_names::AbstractVector,
    constants::AbstractVector{<:Integer}
) -> Any

Builds a grammar based on the base QN grammar adding entity_names and constants to the grammar.

Four constraints are currently included

  1. removing symmetry due to commutativity of +/*/min/max
  2. forbidding same arguments of two argument functions
  3. forbidding trivial inputs (consts and entity values) to floor/ceil
  4. forbidding ceil(floor(_)) and floor(ceil(_))

Create an asynchronous Boolean network (ABN) with the given network, initial_state. The update step for an asynchronous Boolean network is to choose a random node and update its state, one node at a time. The random choice is uniform over all nodes. See abn_step!.

abn_step!(
    model::GraphDynamicalSystems.BooleanNetworks.BooleanNetwork
) -> Any

Update step for an asynchronous Boolean network (see abn). At each step, It selects a node in the network and applies its update function, updating the state for the selected node with the update function's output.
